# 🎙️ VoiceSync

VoiceSync is an AI-powered Text-to-Speech web application that converts text into natural-sounding speech. It provides a clean and responsive interface with secure authentication, multiple voice options, adjustable playback speed, audio download, and real-time playback controls.

## ✨ Features

- 🎙️ Text-to-Speech conversion
- 🔊 Multiple voice options
- ⚡ Adjustable playback speed
- ▶️ Audio playback controls
- 📥 Download generated audio
- 🔐 Secure user authentication
- 📱 Responsive modern UI

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- PostgreSQL
- Supabase Auth
- Supabase Storage
- Vercel

## 🚀 Installation

```bash
git clone https://github.com/kokilavelusami/Voicesync.git
cd Voicesync
npm install
npm run dev
```
